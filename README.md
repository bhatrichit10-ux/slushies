# ClubsAPI Dash
Hii, i built a normal flask app which uses Hack Club ClubsAPI to fetch information about club. You can use this panel to get information about different Clubs

## Stack
HTML <br>
TailwindCSS <br>
Flask <br>

## Routes
This app has 3 routes:
- `/index` or `/` - Main Page
- `/club/<clubname>` - Club Info Page
- `/error` - Error Page
